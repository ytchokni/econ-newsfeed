## Summary

Next.js newsfeed UI with three pages: a chronological publication feed, a researcher directory, and a researcher detail page. Includes SWR data fetching, pagination, and loading/error/empty states.

## Context

The econ-newsfeed project tracks economists' personal websites and detects new publications. The backend API (DESIGN.md Section 4) provides paginated publication and researcher endpoints. This track implements the frontend that consumes those endpoints.

## User Story

As an economics researcher, I want to browse a chronological feed of new publications and view individual researcher profiles so that I can stay up to date without checking dozens of websites.
