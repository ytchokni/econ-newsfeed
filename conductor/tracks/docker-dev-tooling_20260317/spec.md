## Summary

Create Docker containerization (Dockerfile.api, app/Dockerfile, docker-compose.yml) with three services (MySQL, API, frontend) and a Makefile with setup/dev/seed/reset targets for standardized development workflows.

## Context

The econ-newsfeed project is a full-stack application (FastAPI + Next.js + MySQL) that tracks economists' publications. Currently it requires manual setup to run. Docker Compose is listed in the product scope as the target local development and deployment strategy.

## Motivation

The project currently only runs via manual setup. Docker Compose enables one-command startup for the full stack, and the Makefile standardizes common development workflows. This is needed for both local development and eventual production deployment.

- [ ] `.dockerignore` files prevent `.git`, `node_modules`, `.env`, `__pycache__`, `.venv` from being copied into images
